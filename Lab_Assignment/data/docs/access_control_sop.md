# Access Control SOP

- Level 2 access can be granted temporarily to contractors only with a business owner approval and Security approval.
- Emergency Level 2 temporary contractor access expires after 24 hours and requires post-review by Security.
- Level 3 emergency access requires two approvers: the system owner and the Security Manager.
- The final approver for Level 3 emergency access is the Security Manager.
- Level 3 access must be logged with reason, duration, requester, approvers, and revocation time.
- Sources: Access Control SOP.
